Configurar OpenAI API key

Para transcrever os áudios, a chave é armazenada somente na sua máquina via chrome.storage.

Atenção: armazenar a chave no cliente tem riscos. O ideal é usar um backend próprio. Não compartilhe sua chave. [Ver boas práticas](https://help.openai.com/en/articles/5112595-best-practices-for-api-key-safety)
